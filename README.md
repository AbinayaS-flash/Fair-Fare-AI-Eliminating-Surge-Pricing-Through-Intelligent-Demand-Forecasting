Project Name: FairFare AI – AI-Powered Fair Pricing & Demand Prediction System

Problem: In current ride-hailing systems, fares change unpredictably due to surge pricing, making rides expensive and unfair for passengers while driver earnings remain unstable. Users have no transparency on how fares are calculated, leading to distrust in pricing systems and poor user experience.

Solution: FairFare AI is an intelligent pricing system that predicts ride demand in advance and calculates a transparent, stable fare using AI-based demand forecasting. It provides a clear fare breakdown, compares it with traditional surge pricing, and ensures fair compensation for drivers. The system improves affordability, transparency, and earnings stability in mobility services.

Technology Stack: Frontend: Streamlit Backend: Python AI & ML: Scikit-learn, Random Forest Regressor, Pandas, NumPy Visualization: Plotly Data Format: CSV Deployment: Streamlit Cloud / Local VS Code Version Control: Git, GitHub
